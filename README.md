# Система управления орентацией спутника

## Загрузка проекта
Для загрузки проекта ввести команду:
`git clone --recursive git@github.com:FruitEnjoyer/AttitudeControlSystem.git`

## Сборка проекта
Для сборки проекта:
- Установить путь до библиотеки eigen в переменную LIBEIGEN_PATH в CMakeLists.txt
- Собрать стандартным способом через cmake и make

### Требования
- Библиотека линейной алгебры Eigen
`sudo apt install libeigen3-dev`
